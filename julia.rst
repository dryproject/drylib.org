Getting Started
===============

Prerequisites
-------------

- Julia 0.6+

Installation
------------

::

   julia> Pkg.clone("https://github.com/dryproject/DRYlib.jl.git")

Library Usage
-------------

::

   julia> using DRYlib
   julia> const DRY = DRYlib
