Getting Started
===============

Prerequisites
-------------

- `Go <https://en.wikipedia.org/wiki/Go_(programming_language)>`__
  1.9+ (due to use of type aliases)

Installation
------------

.. code-block:: bash

   $ go get github.com/dryproject/drylib.go

Library Usage
-------------

To import the library, do::

   import "github.com/dryproject/drylib.go"
